EJ-CUP — Whistle Live Event System

Live System Custom Software Real-Time

Context

EJ-CUP is a community football tournament requiring precise match control, live score tracking, and reliable timing under real-world event conditions.

Bladio Studio was commissioned to build a dedicated software solution to replace manual workflows and reduce operational risk during live matches.

Problem

  • Manual scoreboards and timers prone to human error
  • No centralized control over match state
  • Live changes needed during ongoing games
  • High pressure environment with zero tolerance for downtime

Solution

Bladio Studio developed Whistle, a custom live event control system designed specifically for football tournaments.

  • Centralized match control interface
  • Real-time score and timer management
  • OBS / broadcast output integration
  • Clear separation between control logic and visual output
  • Fail-safe workflows to minimize operator errors

Outcome

  • Stable operation throughout the entire event
  • No manual scoreboard intervention required
  • Reduced operational stress for event staff
  • Reusable system architecture for future tournaments

Why it matters

This project demonstrates Bladio Studio’s ability to deliver mission-critical software under live conditions.

The same system principles apply to other live environments such as esports events, conferences, monitoring dashboards, and control software.
